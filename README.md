# Calculator App

A simple and responsive Calculator Application developed for the IBM Learning / GitHub course project purpose. This project demonstrates the fundamentals of front-end development, including user interface design, mathematical operations handling, and interactive functionality using modern web technologies.

## Features

* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* Clear and delete functions
* Responsive and clean user interface
* Real-time calculation display
* Beginner-friendly project structure

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

This project was created for educational and learning purposes under the IBM Learning Program. It helps learners understand:

* DOM manipulation
* Event handling
* JavaScript functions
* Responsive UI development
* Basic application logic

* Open `index.html` in your browser
